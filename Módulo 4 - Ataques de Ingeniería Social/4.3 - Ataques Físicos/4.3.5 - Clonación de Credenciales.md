# Clonación de credenciales
Los atacantes pueden realizar diferentes ataques de clonación de credenciales. Por ejemplo, un atacante puede clonar una tarjeta de identificación utilizada para acceder a un edificio. Se pueden utilizar software y hardware especializados para realizar estos ataques de clonación. Los atacantes también pueden utilizar técnicas de ingeniería social para hacerse pasar por empleados o cualquier otro usuario autorizado para ingresar a un edificio simplemente creando su propia tarjeta de identificación e intentando engañar a otros usuarios para que los dejen ingresar al edificio. Esto incluso podría realizarse sin una clonación completa de las capacidades de radiofrecuencia (RF) de una tarjeta de identificación.
