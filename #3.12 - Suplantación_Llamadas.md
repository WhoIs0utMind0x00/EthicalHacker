# Herramientas de suplantación de identidad de llamadas
Puede cambiar fácilmente la información del identificador de llamadas que se muestra en un teléfono. Existen varias herramientas que se pueden utilizar en ataques de ingeniería social.
- _SpoofApp_: Es una aplicación de Apple para iOS y Android que se puede usar para falsificar fácilmente un número de teléfono.
- _SpoofCard_: Es una aplicación de Apple para iOS y Android que puede falsificar un número y cambiar su voz, grabar llamadas, generar diferentes ruidos de fondo y enviar llamadas directamente al correo de voz.
- _Asterisk_: Es una herramienta legítima de administración de voz sobre IP (VoIP) que también se puede utilizar para suplantar el identificador de llamadas.
