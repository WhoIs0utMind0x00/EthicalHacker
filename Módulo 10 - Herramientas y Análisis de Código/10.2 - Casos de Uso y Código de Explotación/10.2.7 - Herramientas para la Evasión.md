# 10.2.7 - Herramientas Comunes para la Evasión
En un compromiso de prueba de penetración, generalmente desea mantener el sigilo e intentar evadir y eludir los controles de seguridad que la organización pueda tener. Se pueden utilizar varias técnicas y herramientas para la evasión, incluidas las siguientes:<br>
+ Veil
+ Tor
+ Cadenas de proxy
+ Cifrado
+ Encapsulación y tunelización mediante DNS y protocolos como NTP

### Veil
__[Veil](https://github.com/Veil-Framework/Veil)__ es un marco que se puede usar con Metasploit para evadir los controles de antivirus y otros controles de seguridad. A continuación, un ejemplo de Veil:<br>
#### Paso 1. Iniciar Veil
Ejecutar el comando `veil`. Después, se muestra el menú Veil.<br>
#### Paso 2. Seleccionar evasión
Para usar Veil para la evasión, seleccione la primera opción (número 1). Veil luego muestra las cargas útiles disponibles y los comando de Veil.<br>
#### Paso 3. Enumerar las cargas útiles
Para enumerar las cargas útiles disponibles, utilice el comando `list`.<br>
#### Paso 4. Instalar una carga útil
Como ejemplo, se utilizará la carga útil de TCP inverso de Meterpreter. Después de seleccionar la carga útil, debe configurar el host local (LHOST) y luego usar el comando `generate` para generar la carga útil.<br>
#### Verificar la ubicación del archivo de carga útil
Una vez generada la carga útil, se muestra otra pantalla. En la cual, se enumeran las ubicaciones del ejecutable de la carga útil, el código fuente y el archivo de recursos de Metasploit.<br>
