# 10.2.5 - Herramientas comunes para ataques de credenciales
Las siguientes son algunas de las herramientas populares que se pueden utilizar para aplicar fuerza bruta, descifrar y comprometer las credenciales de usuario:<br>
+ John the Ripper
+ Caín y Abel
+ Hashcat
+ Hydra
+ Rainbowcrack
+ Medusa y Ncrack
+ CeWL
+ Mimikatz
+ Patator

<hr>

### John the Ripper
__[John the Ripper](https://www.openwall.com/john)__ es una herramienta muy popular para descifrar contraseñas fuera de línea. John the Ripper (o John para abreviar) puede usar patrones de búsqueda y archivos de contraseñas (o listas de palabras) para descifrar contraseñas. Admite diferentes modos de descifrado y comprende muchos formatos de texto cifrado, incluidas varias variantes de DES, MD5 y Blowfish. John the Ripper no admite AES y SHA-2. Para enumerar los formatos admitidos, puede usar el comando john --list = formats. John the Ripper también se puede utilizar para extraer las contraseñas de Kerberos AFS y Windows.<br>

### Cain
__[Cain (o Cain y Abel)](https://sectools.org/tool/cain/)__ es una herramienta heredada que puede utilizarse para "recuperar" contraseñas de sistemas basados en Windows. Se puede utilizar para descifrar y recuperar credenciales de usuario mediante la realización de capturas de paquetes (rastreo); descifrar contraseñas cifradas mediante ataques de diccionario, fuerza bruta y criptoanálisis; y el uso de muchas otras técnicas.<br>

### Hashcat
__[Hashcat](https://hashcat.net/hashcat)__ es otra herramienta para descifrar contraseñas muy popular entre los evaluadores de penetración. Le permite utilizar unidades de procesamiento gráfico (GPU) para acelerar el proceso de descifrado de contraseñas.<br>
__NOTA__: Hashcat viene con Kali Linux y otras distribuciones de Linux para pruebas de penetración.<br>

### Hydra
Hydra es otra herramienta que puede utilizarse para adivinar y descifrar credenciales. Hydra se usa generalmente para interactuar con un servidor de la víctima (por ejemplo, servidor web, servidor FTP, servidor SSH, servidor de archivos) y probar una lista de combinaciones de nombre de usuario / contraseña. Por ejemplo, supongamos que sabe que el nombre de usuario de un usuario de FTP es omar. A continuación, puede probar con un archivo que contiene una lista de contraseñas en un servidor FTP (10.1.2.3). Para lograr esto, utilice el siguiente comando:<br>
```bash
    hydra -l omar -P passwords.txt ftp://10.1.2.3
```
El archivo passwords.txt contiene una lista de contraseñas comunes para probar. Además, puede crear un archivo que tenga una combinación de nombres de usuario y contraseñas y usar Hydra para realizar un ataque de fuerza bruta, de la siguiente manera:
```bash
    hydra -L logins.txt -P passwords.txt ftp://10.1.2.3
```

### Rainbowcrack
Los atacantes pueden utilizar _rainbow tables_, tablas precalculadas para revertir las funciones hash criptográficas, para acelerar el descifrado de contraseñas. Es posible utilizar una tabla arcoíris para derivar una contraseña mirando el valor hash.<br> La herramienta __[RainbowCrack](http://project-rainbowcrack.com/)__ se puede utilizar para automatizar el descifrado de contraseñas mediante tablas de arcoíris.<br>
__NOTA__: El siguiente sitio web incluye una lista de tablas arcoíris que se pueden usar con RainbowCrack: _http://project-rainbowcrack.com/table.htm_.<br>

### Medusa y Ncrack
Las herramientas __*Medusa*__ y __[Ncrack](https://nmap.org/ncrack)__, que son similares a Hydra, se pueden utilizar para realizar ataques de fuerza bruta a las credenciales contra un sistema. Puede instalar Medusa mediante el comando `apt install medusa` en un sistema basado en Debian (como Ubuntu, Kali Linux o Parrot OS). Puede descargar Ncrack desde su sitio web o con el comando `apt install ncrack`.<br>

### CeWL
__[CeWL](https://digi.ninja/projects/cewl.php)__ es una gran herramienta que puede utilizarse para crear listas de palabras. Puede utilizar CeWL para rastrear sitios web y recuperar palabras.<br>

### Mimikatz
__[Mimikatz](https://github.com/gentilkiwi/mimikatz)__ es una herramienta que muchos evaluadores de penetración y atacantes (e incluso malware) utilizan para recuperar hashes de contraseñas de la memoria. También es una útil herramienta posterior a la explotación. Metasploit también incluye Mimikatz como script de Meterpreter para facilitar la explotación sin la necesidad de cargar ningún archivo en el disco del host comprometido. Puede obtener más información sobre la integración de Mimikatz y Metasploit en _https://www.offsec.com/metasploit-unleashed/mimikatz/_.<br>

### Patator
__[Patator](https://github.com/lanjelot/patator)__ es otra herramienta que se puede utilizar para ataques de fuerza bruta en enumeraciones de nombres de usuario SNMPv3, contraseñas de VPN y otros tipos de ataques de credenciales.<br>
