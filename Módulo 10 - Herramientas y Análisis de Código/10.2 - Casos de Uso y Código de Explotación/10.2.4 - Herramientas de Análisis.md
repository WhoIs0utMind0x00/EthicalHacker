# 10.2.4 - Herramientas comunes para el Análisis de Vulnerabilidades
Existen numerosas herramientas de análisis de vulnerabilidades, incluidos escáneres de vulnerabilidades de código abierto y comerciales, así como servicios y herramientas basados en la nube. A continuación, se enumeran algunos de los escáneres de vulnerabilidades más populares:<br>
+ OpenVAS
+ Nessus
+ Nexpose
+ Qualys
+ SQLmap
+ Nikto
+ OWASP Zed Attack Proxy (ZAP)
+ w3af
+ DirBuster
+ Brakeman
+ Escáneres de Protocolo de Automatización de Contenido de Seguridad Abierto (SCAP)
+ Wapiti
+ Suite Scout
+ WPScan (escáner de Wordpress)

__NOTA__: __[Brakeman](https://brakemanscanner.org/)__, __[SCAP](https://www.open-scap.org/)__, __[Wapiti](https://wapiti.sourceforge.io/)__, __[Scout Suite](https://github.com/nccgroup/ScoutSuite)__ y __[WPScan](https://wpscan.org/?)__ no se analizan más en este curso. Para obtener más información, visite sus sitios web.<br>
OWASP enumera herramientas de análisis de vulnerabilidades adicionales no cubiertas en este módulo en *https://www.owasp.org/index.php/Category:Vulnerability_Scanning_Tools*.<br>
<hr>

### OpenVAS
[OpenVAS](https://www.openvas.org) es un escáner de vulnerabilidades de código abierto creado por Greenbone Networks. El marco de OpenVAS incluye varios servicios y herramientas que le permiten realizar un análisis detallado de vulnerabilidades en hosts y redes.<br>
#
### Nessus
El escáner __[Nessus](https://www.tenable.com/downloads/nessus)__ de Tenable tiene varias características que le permiten realizar un monitoreo continuo y análisis de cumplimiento.<br>
__NOTA__: Tenable también tiene una solución basada en la nube llamada __[Tenable.io](https://www.tenable.com/products/tenable-io)__.<br>
#
### Nexpose
Nexpose es un escáner de vulnerabilidades creado por Rapid7 que es muy popular entre los evaluadores de penetración profesionales. Admite integraciones con otros productos de seguridad.<br>
__NOTA__: Rapid7 también tiene varias soluciones de análisis de vulnerabilidades que se utilizan para la administración de vulnerabilidades, la supervisión continua y el ciclo de vida de desarrollo seguro.<br>
#
### Qualys
__[Qualys](https://www.qualys.com)__ es una empresa de seguridad que creó uno de los escáneres de vulnerabilidades más populares de la industria. También tiene un servicio basado en la nube que realiza monitoreo continuo, administración de vulnerabilidades y verificación de cumplimiento. Esta solución en la nube interactúa con agentes en la nube, escáneres virtuales, dispositivos de escáner y escáneres de Internet.<br>
Herramientas como Qualys y Nessus también proporcionan funciones que se pueden utilizar para el cumplimiento de la configuración.
#
### SQLmap
__[SQLmap](http://sqlmap.org)__ a menudo se considera una vulnerabilidad web y una herramienta de inyección de SQL. Ayuda a automatizar la enumeración de aplicaciones vulnerables, así como el aprovechamiento de las técnicas de inyección SQL que aprendió en el Módulo 6, “Aprovechamiento de las vulnerabilidades basadas en aplicaciones”.<br>
#
### Nikto
Nikto es un escáner de vulnerabilidades web de código abierto que se puede descargar de _https://github.com/sullo/nikto_. Se puede acceder a la documentación oficial de Nikto en _https://cirt.net/nikto2-docs_.<br>
#
### OWASP Zed Attack Proxy (ZAP)
Según OWASP, OWASP Zed Attack Proxy (ZAP) "es una de las herramientas de seguridad gratuitas más populares del mundo y la mantienen activamente cientos de voluntarios internacionales". Muchos ingenieros de seguridad ofensivos y defensivos de todo el mundo utilizan ZAP, que no solo proporciona capacidades de análisis de vulnerabilidades web, sino que también se puede utilizar como un proxy web sofisticado. ZAP viene con una API y también se puede utilizar como fuzzer. Puede descargar y obtener más información sobre OWASP ZAP en https://www.owasp.org/index.php/OWASP_Zed_Attack_Proxy_Project.<br>
#
### w3af
Otro escáner popular de vulnerabilidades de aplicaciones web de código abierto es w3af. w3af se puede descargar de https://github.com/andresriancho/w3af y su documentación se puede obtener en https://docs.w3af.org/en/latest/.<br>
La herramienta w3af tiene un menú de complementos que le permite configurar y habilitar mangle, crawl, bruteforce, audit y otros complementos.<br>
#
### DirBuster
DirBuster es una herramienta que fue diseñada para aplicar fuerza bruta a nombres de directorios y archivos en servidores de aplicaciones web. DirBuster es actualmente un proyecto inactivo y su funcionalidad se ha integrado y mejorado en OWASP ZAP como complemento.<br>
__NOTA__: DirBuster es una aplicación Java diseñada para aplicar la fuerza bruta a directorios y nombres de archivos en servidores web o de aplicaciones. A menudo, lo que parece un servidor web con una instalación predeterminada en realidad tiene páginas y aplicaciones ocultas. DirBuster intenta encontrarlos. Dos alternativas adicionales a DirBuster son [gobuster](https://github.com/OJ/gobuster) y [ffuf](https://github.com/ffuf/ffuf). Tenga en cuenta que las herramientas de esta naturaleza a menudo son tan buenas como el directorio y las listas de archivos con las que vienen.
