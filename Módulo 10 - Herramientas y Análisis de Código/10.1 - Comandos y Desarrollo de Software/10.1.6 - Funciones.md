# 10.1.6 - Funciones
Una función es un bloque de código que es muy útil cuando necesita ejecutar tareas similares una y otra vez. Una función se ejecuta solo cuando se le llama.<br>
Los siguientes son algunos recursos que puede usar para aprender sobre funciones en diferentes lenguajes de programación:<br>
+ [Funciones de Python](https://www.tutorialspoint.com/python/python_functions.htm)
+ [Funciones de JavaScript](https://www.w3schools.com/js/js_function_definition.asp)
+ [Funciones de Bash](https://linuxize.com/post/bash-functions/)
+ [Funciones de PowerShell](https://docs.microsoft.com/en-us/powershell/scripting/learn/ps101/09-functions?view=powershell-7.1)
