# 10.1.4 - Bibliotecas
Una biblioteca es una colección de recursos que los programas pueden reutilizar. Las bibliotecas pueden incluir lo siguiente:<br>
+ Código escrito previamente
+ Información de configuración
+ Subrutinas
+ Documentación e información de ayuda
+ Plantillas de mensajes
+ Clases

__CONSEJO__: Cada lenguaje de programación admite un conjunto de bibliotecas estándar y de terceros. Por ejemplo, el siguiente sitio web describe la biblioteca estándar de Python: __https://docs.python.org/3/library__.<br>
