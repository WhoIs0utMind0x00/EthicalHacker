# 10.1.3 - Estructuras de Datos
Las siguientes son las estructuras de datos más utilizadas en los lenguajes de programación:<br>

+ __Notación de objetos de JavaScript (JSON)__: JSON es un formato ligero para almacenar y transportar datos que es fácil de entender. Es la estructura de datos más común en las API RESTful y muchas otras implementaciones. Puede aprender de forma interactiva en __[JSON en w3schools](https://www.w3schools.com/whatis/whatis_json.asp)__.
+ __Matrices__: Una matriz es un valor especial que contiene más de un valor a la vez.
+ __Diccionarios__: Un diccionario es una colección de valores de datos que se ordenan mediante un para **_clave/valor_**. El siguiente es un ejemplo de un diccionario en Python:<br>

    ```Python
        dict = {'ID': '0X0FVCKYOV', 'Nombre': 'Abib Pérez', 'Nombre de Usuario': 'Mr_3xpl0it'}
    ``` 
+ __Valores separados por comas (CSV)__: Un archivo CSV es un archivo de texto sin formato que contiene datos delimitados por comas (,) y, a veces, tabulaciones u otros caracteres, como punto y coma (;).
+ __Listas__: Una lista es una estructura de datos en lenguajes de programación que contiene una estructura ordenada de elementos. El siguiente es un ejemplo de una lista en Python
    ```Python
        cves = ['CVE-2022-0945', 'CVE-2023-1234', 'CVE-2022-0987']
    ```
+ __Árboles__: Un árbol es una estructura de datos no lineal representada mediante nodos en un modelo jerárquico. El siguiente sitio incluye ejemplos de árboles (o árboles binarios) en Python donde puede aprender a interactuar con el código fuente: __https://www.educative.io/edpresso/binary-trees-in-python__.<br>
