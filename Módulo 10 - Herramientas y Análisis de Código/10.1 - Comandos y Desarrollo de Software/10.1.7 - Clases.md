# 10.1.7 - Clases
Una clase es una plantilla de código que se puede utilizar para crear diferentes objetos. Proporciona valores iniciales para variables miembro y funciones o métodos. En los lenguajes de programación orientados a objetos, como Java, Python y C++, numerosos componentes son objetos, incluidas las propiedades y los métodos. Una clase es como un plan para crear objetos.<br>
__CONSEJO__: El siguiente sitio web incluye varios ejemplos de clases de Python: **https://www.w3schools.com/python/python_classes.asp**. Para obtener más información sobre las clases de JavaScript, consulte **https://www.w3schools.com/js/js_class_intro.asp**.
