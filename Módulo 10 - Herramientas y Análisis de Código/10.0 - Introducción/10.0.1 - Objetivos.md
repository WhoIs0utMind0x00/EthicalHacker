# 10.0.1 - ¿Porqué debería tomar este módulo?

### Soluciones de seguridad Protego
Creo que admitirá que hay mucho que recordar si va a trabajar de manera eficaz como evaluador de penetración. Parece desalentador para alguien que recién comienza. Sin embargo, esto es válido para cualquier campo. Parece mucho hsata que se pone a trabajar y utiliza sus conocimientos y habilidades todos los días.<br>
Las empresas de pruebas de penetración no utilizan todo lo que aprendió en este curso. Cada uno tiene sus propias herramientas, scripts, lenguajes de scripts y formatos de documentos preferidos. Aprenderá sobre esto en el trabajo.<br>
Sabemos que es abrumador, por lo que hemos incluido este módulo para que lo consulte cuando necesite trabajar con código o encontrar la herramienta adecuada para el trabajo. Es bueno estar familiarizado con todo en este módulo, pero no se preocupe si tiene problemas para recordar detalles sobre cada herramienta mencionada aquí. Realmente, solo concéntrese en qué herramientas son útiles para qué tareas generales. Entonces, siempre puede buscar lo que no recuerde.<br>
<hr>
Las pruebas de penetración y el hacking ético no se limitan a herramientas y scripts interesantes; requieren buenas metodologías, pensar como un atacante y habilidades técnicas avanzadas. Aún así, las herramientas pueden ayudar a acelerar un compromiso de pruebas de penetración y ayudarlo a escalar. En este módulo, también aprenderá sobre diferentes casos de uso de herramientas de pruebas de penetración. También aprenderá a analizar el resultado de las herramientas de pruebas de penetración más populares para realizar evaluaciones informadas. Al final del módulo, aprenderá cómo aprovechar el shell de Bash, Python, Ruby, PowerShell, Perl y JavaScript para realizar secuencias de comandos básicas.<br>
