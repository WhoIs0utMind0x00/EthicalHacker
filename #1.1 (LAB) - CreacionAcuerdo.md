# Creación de un acuerdo de prueba de penetración

## Objetivos
- Completa un acuerdo de pruebas de penetración simple

## Trasfondo / Escenario

Un acuerdo de pruebas de penetración es un contrato legalmente vinculante entre el cliente o el evaluador de penetración. El acuerdo define todos los términos y condiciones requeridos para el ejercicio de prueba de penetración. El acuerdo incluirá elementos acordados mutuamente por ambas partes. Puede contener elementos, como la fecha de inicio de las pruebas, el alcance del trabajo, el acuerdo de nivel de servicio, la posible fecha de finalización de pruebas, etc. También se incluirán en el contrato otros términos y condiciones, así como detalles de precios .

Imagine que una empresa lo contrata para realizar las pruebas. Deberá redactar un acuerdo de pruebas de penetración entre su empresa y el cliente. En una situación real, es probable que consulte con un abogado especializado en dichos contratos porque es muy importante comprenderlos y la información que contienen.

## Recursos necesarios

- Computadora personal o dispositivo móvil con acceso a Internet

## Parte 1: Completa un acuerdo de pruebas de penetración simple
__Paso 1: Investiga los acuerdos y contratos de pruebas de penetración__

- Realiza una búsqueda de contratos de pruebas de penetración, extractos de trabajo y acuerdos.
- Encuentra varios ejemplos de acuerdos de pruebas de penetración y revisa la información contenida en cada sección.

__Paso 2: Desarrolla el acuerdo de pruebas de penetración__

Con la información que recopilaste de la investigación, crea tu propio acuerdo de prueba de penetración para tu empresa. Como mínimo, el acuerdo debe incluir las secciones que se enumeran a continuación:

1. __Partes del acuerdo__ <br> *(resalta los detalles personales y la información de las partes involucradas)*
```
La sección debe indicar claramente el nombre, la dirección y la información de contacto de la empresa cliente y de la empresa de pruebas de penetración 
```
2. __Alcance del trabajo__ <br> *(definir las obligaciones y responsabilidades de cada parte)*
- *El evaluador de penetración acepta:*
```
  Esta sección debe enumerar las responsabilidades y obligaciones del evaluador de penetración. Esto incluirá cosas como las pruebas que se realizarán, qué acciones se tomarán si se encuentran vulnerabilidades y qué resultados puede esperar el cliente.
``` 
- *El cliente acepta:* 
```
Esta sección enumera las responsabilidades y obligaciones del cliente. Esto incluirá cosas como obtener los documentos de consentimiento necesarios, realizar copias de respaldo de los datos críticos y proporcionar las adaptaciones adecuadas.
```
3. __Plazo__ <br> *(Establezca una línea de tiempo específica para las pruebas de penetración)*.
```
Esta sección incluye cosas como una línea de tiempo para los hitos del proyecto, plazos para las fases del proyecto. Las fases pueden incluir planificación, ejecución, análisis, presentación, etc.
```
4. __Tarifas, facturación y detalles de pago__ <br> *(defina cómo y cuándo se realizarán los paos y aborde cómo se obtienen los materiales y los equipos)*.
```
Esta sección define cómo y cuándo se realizan los pagos, por ejemplo, una suma global o en partes. Esta sección también puede incluir cómo se obtienen los materiales y equipos de prueba necesarios.
```
5. __Rescisión del contrato__ <br> *(define las circunstancias que provocarían la rescisión anticipada del contrato)*.
```
Esta sección incluye circunstancias y situaciones que pueden conducir a la terminación anticipada del contrato por cualquiera de las partes. Aborda el derecho de la empresa de pruebas de penetración o del cliente a finalizar la asociación por cuestiones como el incumplimiento del pago de las tarifas o las pruebas no realizadas correctamente. Debe explicar el curso de acción a tomar en caso de cancelación anticipada.
```

__Paso 3: Bonificación - Secciones adicionales del acuerdo__

Podría incluir garantía, exención de responsabilidad, limitación de responsabilidad y resolución de disputas.
