# 7.1.3 - Escalamiento de Privilegios
El escalamiento de privilegios es el acto de aprovechar un error o falla de diseño en una aplicación de software o firmware para obtener acceso a recursos de una aplicación o un usuario que normalmente habrían estado protegidos. Esto da como resultado que un usuario obtenga privilegios adicionales más allá de lo que pretendía originalmente el desarrollador de la aplicación (por ejemplo, un usuario regular que obtiene el control administrativo o un usuario en particular puede leer el correo electrónico de otro usuario sin autorización).<br>
El desarrollador original no tiene la intención de que el atacante obtenga mayores niveles de acceso, pero probablemente no aplique una política de necesidad de conocer correctamente o no haya validado el código de la aplicación de manera adecuada. Los atacantes se aprovechan de esto para obtener acceso a áreas protegidas de los sistemas operativos o a las aplicaciones. Los desbordamientos de búfer también se utilizan en computadoras con Windows para elevar los privilegios. Para eludir la administración de derechos digitales (DRM) en juegos y música, los atacantes usan un método conocido como **_jailbreaking_**, que es otro tipo de escalamiento de privilegios, más comúnmente encontrado en dispositivos móviles basados en iOS de Apple. El malware intenta aprovechar las vulnerabilidades de escalamiento de privilegios, si existen en el sistema. También se puede intentar el escalamiento de privilegios en dispositivos de red. En general, la solución es simplemente actualizar el dispositivo y buscar actualizaciones periódicamente.<br>
Los siguientes son algunos tipos diferentes de escalamiento de privilegios:<br>

### Escalamiento de privilegios vertical
Este tipo de escalamiento de privilegios, también denominado elevación de privilegios, se produce cuando un usuario con menos privilegios accede a funciones reservadas para usuarios con más privilegios (por ejemplo, un usuario estándar que accede a las funciones de un administrador). Para protegerse contra esta situación, debe actualizar el firmware del dispositivo de red. En el caso de un sistema operativo, debe actualizarse nuevamente. El uso de algún tipo de sistema de control de acceso, por ejemplo, Control de Cuentas de Usuario (UAC), también es recomendable.<br>

### Escalamiento de privilegios horizontal
Este tipo de escalamiento de privilegios se produce cuando un usuario normal accede a funciones o contenido reservado para otros usuarios normales. Esto se puede hacer mediante la piratería o mediante una persona que se acerca a la computadora de otra y simplemente lee su correo electrónico. Haga que los usuarios bloqueen siempre la computadora (o desconecten la sesión) cuando no estén físicamente en su escritorio.<br>
